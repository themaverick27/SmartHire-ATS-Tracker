# SmartHire – AI-Powered ATS Resume Analyzer

SmartHire is an **AI-powered ATS (Applicant Tracking System) Resume Analyzer** built using **Python, Streamlit, and Gemini AI**.  
It analyzes resumes against job descriptions, extracts keyword insights, and provides actionable suggestions to improve **interview shortlisting chances**.

---

## Features

- Resume analysis and parsing  
- Job Description vs Resume matching  
- Keyword gap & keyword match insights  
- Gemini AI–powered evaluation  
- ATS-style resume assessment  
- Interactive UI using Streamlit  

---

## How It Works

1. Upload or paste the resume content  
2. Paste the job description  
3. Gemini AI analyzes both inputs  
4. The system provides:
   - Keyword match & gaps  
   - ATS-style insights  
   - Resume improvement suggestions
